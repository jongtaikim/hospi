동적 족저압 측정 시스템(Rs SCAN)

HP206

족압 측정(Foot pressure measurement)시스템은 병원, 신발공학 연구소, 스포츠 관련 연구분야등에서
발바닥의 정적/동적 압력 분포를 측정하고, 다양한 분석 소프트웨어 모듈을 통해 발바닥의 압력 편중으로 인한
질환 파악, 발 동작 분석, 신발의 인체공학적 설계등에 사용한다.

하드웨어
Plate size :
1) 0.5m (50cm X 40cm : 4096 sensors)
2) 1m (100cm X 40cm : 8192 sensors)
3) 2m (200cm X 40cm : 16384 sensors)
4) USB2 시스템 (0.5 plate : 50cm X 40cm : 4096 sensors)

소프트웨어
- Client 데이터 베이스 : 피검자의 과거/현재 측정 데이터를 자동으로 저장하여 쉽게 비교분석
- 정적(Static) 측정 데이터 제공 :
Display of partitioning of body weight over 4 quadrants Display of center of gravity
- 동적(Dynamic) 측정 데이터 제공 :
걷기/런닝 측정, 맨발/신발착용 측정, 왼발/오른발 자동감지, 실시간 압력 전송, 압력중심(Center of Pressure)
이동선, 2D/3D, 동기화, Impulse&Roll off 데이터 전송
- 비교 스크린(Comparison sereen)제공 : 왼쪽/오른쪽/과거/현재의 측정 데이터 비교분석
- 발란스 스크린 :
Heel rotation, Foot balance, Medial forefoot balance, Forefoot rotation, Forefoot balance, Metatarsal loading
- Dynamic 스크린 : 측정 상태의 데이터의 실시간 표현
- 힘/압력 그래프 : 족저압의 이동 및 변화를 쉽게 관찰 할 수 있도록 부분영역별 데이터 그래프 표현
- 타이밍 스크린 : 보행 사이클 관련하여 의미있는 시간 데이터의 변화 추이를 제공
