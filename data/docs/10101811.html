전산화 인지평가 시스템(CNT40)

HO524

CNT40 (Compyterized Neuro - Cognitive Function Test Software)

특징
- CNT40은 한글화된 자극을 이용하여 다양한 신경인지기능을 검사
- 검사의 지시문이 화면에 문자로 표시되며 동시에 음성으로 안내되어 검사의 편의성을 도모
- 주의력, 기억력, 기억-운동 협응능력, 종합적 인지사고 능력 등 다양한 신경인지기능을 검사
- 종이와 연필 또는 카드를 사용하는 지필검사와 비교하여 검사시간을 단축
검사항목
- Digit Span Test(숫자따라하기 검사)
- Visual Span Test(시간 단기기억 검사)
- Auditory C.P.T(언어 지속검사)
- Auditory Controlled C.P.T(조건 언어지속 검사)
- Visual C.P.T(시각 지속 검사)
- Visual Controlled C.P.T(조건 시각지속 검사)
- Crossover Test
- Modaility Shift Test
- Word-Color Test(단어 채색 검사)
- Trial Making Test(선로 잇기 검사)
- Card Sorting Test(카드 분류 검사)
- Verbal Learning Test(언어 기억 검사)
- Visual Learning Test(시각 기억 검사)
- Hypothesis Formation Test(개념 형성 검사)
- Finger Tapping Test(수지력 검사)
- Raven's CPM(색채 도형 지능 검사)
- Raven's SPM(표준 도형 지능 검사)
- Color Trail Making Test(색체 선로 잇기 검사)
효과
기억력 검사, 주의력 검사, 고위 인지기능 검사, 시운동 협응 능력등을 검사

적용 범위
뇌손상, 치매, 약물중독, 알코올중독, 정신분열, 조울병, 우울증, 강박장애, 학습장애, 대인공포증 등
