완전 무선 EMG 시스템(Zero-Wire EMG)

HP205

특징 및 사양
- 완전 무선(Zero wire)으로 모션의 제한성 없음
- 초경량 송신기(12g)
- 동작분석 시스템과 간편한 호환성
- NORAXON EMG 분석 소프트웨어 사용(신뢰성이 있는 데이터의 real-time 분석)
- 대칭분석(Symmetry Protocol) / 협응 능력 분석(Coordinate protocol)
- 피드백 훈련 / Frequency / Fatigue 분석
- 등속성 장비 / 동작분석 장비와 호환
- 각각의 전극에서 데이터 전송(RF Transmis)
- 8시간 연속 측정 가능한 배터리 성능
- 반경 20M 측정 범위
- PDA 모니터링 지원
- 비디오 동시 캡쳐 기능
